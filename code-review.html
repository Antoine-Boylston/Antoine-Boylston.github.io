CS-499 Capstone Code Review

This page highlights my CS-499 Computer Science Capstone code review. In this video, I introduce the original artifact, explain the purpose of the project, and discuss the enhancement plan across software engineering and design, algorithms and data structures, and databases.

Overview

The Capstone project is designed to demonstrate growth throughout the Computer Science program by selecting prior artifacts and enhancing them in meaningful ways. My code review focuses on the original artifact, its current functionality, and the technical improvements planned for each required category.

Enhancement Areas

  • Software Engineering and Design: Refactoring the original application into a more modular, maintainable structure.
  • Algorithms and Data Structures: Enhancing functionality through improved logic and structured problem-solving.
  • Databases: Converting the portfolio Projects section into a dynamic, database-driven feature using AWS services.

Reflection

This code review represents more than just a walkthrough of a project. It reflects my ability to evaluate an existing solution, identify opportunities for improvement, and communicate technical decisions clearly. It also connects my past coursework to the broader technical direction I’ve been building through hands-on projects and continued learning.
