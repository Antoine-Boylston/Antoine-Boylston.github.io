The following artifacts demonstrate my development across key areas of computer science, including software engineering, algorithms and data structures, and database design. Software Engineering: Modular Refactor

Dragon King Assassin is a text-based adventure game originally developed as a procedural Python script. It was enhanced through a full refactor into a modular architecture, separating core components such as the game engine, player model, and command parser. This transformation improved code organization, maintainability, and scalability while demonstrating modern software engineering practices.

Algorithms & Data Structures: Graph & BFS Implementation

This enhancement expands Dragon King Assassin by introducing algorithmic problem-solving techniques. The game’s structure was transformed into a graph representation, and a breadth-first search (BFS) algorithm was implemented to power a dynamic “Hint” system. This demonstrates the use of data structures, graph traversal, and efficient pathfinding to enhance gameplay functionality.
